유경식
  • 회원
    1. 이벤트

    회원들이 진행중인 이벤트, 종료된 이벤트, 당첨자 발표를 확인하는 화면

    • 진행중인 이벤트 조회
      • 이벤트 이미지 클릭 시 이벤트 상세 페이지로 이동
      • 이벤트 상세 페이지에서 이벤트 참여 버튼 클릭 시 이벤트 참여 가능
      • 중복 및 비로그인 시 참여 불가
    • 종료된 이벤트 조회
      • 이벤트 기간이 끝난 이벤트 조회
    • 당첨자발표 조회
      • 종료된 이벤트에 대한 당첨자 발표

  • 관리자
    1. 이벤트 관리

    이벤트를 관리하는 화면

    • 이벤트 생성
    • 이벤트 당첨자 확인
    • 이벤트 상세 보기
    • 이벤트 수정
    • 이벤트 삭제
    • 이벤트 검색

    2. 이벤트 당첨자리스트 관리

    이벤트 당첨자리스트를 관리하는 화면

    • 이벤트 당첨자 리스트 생성
      • 이벤트에 참여한 회원들을 랜덤으로 돌려서 뽑아 당첨자를 정함
    • 이벤트 당첨자리스트 상세 보기
    • 이벤트 당첨자 리스트 수정
      • 당첨자 수정은 불가, 당첨자 리스트 명만 수정 가능
    • 이벤트 당첨자 리스트 삭제
    • 이벤트 당첨자 리스트 검색

    • 멘토링 공고 질문 답변 조회

    • 멘토링 공고 질문 답변 삭제

    3. 멘토링 관리 (멘티)

    회원(멘티)이 결제한 멘토링 목록을 조회하고 관리하는 화면

    • 결제 내역 조회
    • 후기 등록
    • 회원(멘티)가 작성한 후기 목록 조회
    • 후기 수정

    4. 멘토링 관리 (멘토)

    회원(멘토)이 신청 받은 멘토링 목록을 조회하고 정산을 신청하는 화면

    • 신청 받은 내역 조회
    • 정산 신청
    • 정산 내역 조회
  • 관리자
    1. 멘토링 후기 관리

    2. 정산 및 결제 관리

    • 명예 멘토 조회
