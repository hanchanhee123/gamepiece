    • 진행중인 이벤트 조회
      • 이벤트 이미지 클릭 시 이벤트 상세 페이지로 이동
      • 이벤트 상세 페이지에서 이벤트 참여 버튼 클릭 시 이벤트 참여 가능

    • 종료된 이벤트 조회
      • 이벤트 기간이 끝난 이벤트 조회
    • 당첨자발표 조회
      • 종료된 이벤트에 대한 당첨자 발표

      • 이벤트에 참여한 회원들을 랜덤으로 돌려서 뽑아 당첨자를 정함
    • 이벤트 당첨자리스트 상세 보기
    • 이벤트 당첨자 리스트 수정
      • 당첨자 수정은 불가, 당첨자 리스트 명만 수정 가능
    • 이벤트 당첨자 리스트 삭제
    • 이벤트 당첨자 리스트 검색

이재환
  • 회원
    1. 게임 목록 조회

    사용자가 플랫폼 별 게임 목록을 조회하고 검색할 수 있는 화면

    • 플랫폼 별 게임 목록 조회
    • 게임 검색

    2. 게임 상세 정보 및 리뷰 조회, 작성

    사용자가 클릭한 게임의 상세정보와 게임에 작성되있는 리뷰 목록을 조회하고, 사용자가 직접 리뷰를 작성할 수 있는 화면

    • 게임 상세 정보 조회
    • 게임에 작성된 리뷰 목록 조회
    • 리뷰 작성

    3. 장바구니

    사용자가 구매를 희망하는 게임들을 모아 한번에 확인할 수 있게 하는 화면

    • 게임의 예상 총 가격 조회
    • 장바구니 목록 전체 삭제
    • 장바구니 목록 선택 삭제
    • 결제 페이지 이동

    4. 결제 페이지

    사용자가 게임을 구매할 떄 원하는 결제 방식을 결정하고 결제를 진행 할 수 있는 화면

    • 결제할 게임 목록
    • 결제될 게임 금액
    • 결제 방식 선택
    • 결제 진행

    • 메인화면 이벤트 리스트 조회

    유저들이 작성한 카테고리, 제목, 내용, 파일, 이미지를 입력받아 게시판 데이터에 전달한다.

    유저들이 작성한 게시판의 내용을 보여준다.

    • 좋아요, 싫어요 설정
    • 덧글 작성
    • 파일 다운로드
    • 유저 신고
    • 게시물 수정, 삭제

    3. 문의사항

    • 유저들이 작성한 댓글 조회

    4. 신고처분

    신고글을 접수받아 해당 회원에게 처분을 내림
